# BBJin Web - 全球貿易數位化領航者

這是一個極具科技感、高質感的黑底商務風格網站，整合了貿易小工具與多人博弈遊戲。

## 功能特點
- **CCC Code 編碼小工具**：快速校驗與轉換。
- **海關編碼查詢**：即時檢索全球 HS Code。
- **Pandora's Arcade**：多人即時博弈遊戲「潘朵拉魔盒」。

## 頁面清單
- `index.html` - 官方旗艦首頁
- `tools.html` - CCC Code 工具頁
- `hscode.html` - 海關查詢頁
- `game.html` - 遊戲專區
- `style.css` - 核心視覺樣式表
